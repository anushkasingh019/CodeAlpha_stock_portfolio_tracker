# Stock Portfolio Tracker

## Description
A simple Python-based Stock Portfolio Tracker that calculates the total investment value of a portfolio using predefined stock prices. Users can enter stock names and quantities, and the program computes the overall investment amount.

## Features
- User-friendly console interface
- Predefined stock prices using a dictionary
- Accepts stock names and quantities as input
- Calculates individual stock investment values
- Displays total portfolio investment value
- Uses basic Python concepts and data structures

## Technologies Used
- Python
- Dictionary
- Loops
- Conditional Statements
- Input/Output Operations

## Learning Outcomes
This project demonstrates:
- Working with dictionaries in Python
- Using loops and conditional statements
- Taking user input
- Performing arithmetic calculations
- Building simple command-line applications
